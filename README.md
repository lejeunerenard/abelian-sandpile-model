# Abelian Sandpile Model
Simulate infinite fountains in a [ASM](http://en.wikipedia.org/wiki/Abelian_sandpile_model) simulation. Inspired by [Sablo](http://www.disportium.com/index.html#sablo).

![](screenshot.png)

## Install

   ```
npm i
   ```

## Play

   ```
node index.js
   ```

## Develop

   ```
gulp
   ```
